Set up the Android client

  1. Install WireGuard from Google Play (or F-Droid).
  2. Tap ＋ in the app.
  3. Choose Scan from QR code and point it at the code, or Import from file or archive and pick the downloaded .conf.
  4. Name the tunnel {{ interface }} and toggle it on.

The same .conf works for iOS (Scan/Import), macOS, and Linux (wg-quick up ./{{ node.slug }}-{{ interface }}.conf).
